# My Favorite Quoe
## Introduction
This project is an android application that you can save your own quotes in it, or get a random one and save it.
## Features
1. Viewing your quotes
2. adding your own quotes
3. get a random qoute from internet
4. save the random quote
5. set one favorite quote
6. clear all quotes
7. export all quotes
8. change your mode between light mode and dark mode
##Permissions needed
1. internet permission, to get the random quote
2. accessing the download folder, for exporting the quotes
